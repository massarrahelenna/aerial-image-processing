# 🛰️ Aerial Image Processing

Codes and experiments from the **Intelligent Processing of Aerospace Images** course. This repository covers practical classes, decision tree implementations, result analysis, and the final project.

---

## 📁 Repository Structure

```
aerial-image-processing/
├── Aula-Pratica/              # Hands-on class exercises and experiments
├── Códigos-AV1/               # Code used in the first assessment (AV1)
├── Decision_Trees/            # Decision Tree implementations and analysis
├── Resultados/                # Output images and experiment results
├── TaticView/                 # Tactical visualizations and data analysis
├── Trabalho_Final/            # Final course project
└── README.md
```

---

## 🎯 About the Course

The course covers **computer vision** and **machine learning** techniques applied to the processing and analysis of aerospace imagery, such as satellite images and remote sensing data.

---

## 🧠 Technologies & Concepts

- **Python** — primary language for all experiments
- **HTML** — result visualization and reporting
- Decision Trees — image data classification
- Aerospace image processing and analysis
- Performance metrics and result evaluation
